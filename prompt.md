Thực hiện yêu cầu sau: 
Xây dựng trang đánh cờ cho trang web "Chess 64 Squares" sử dụng React Typescript + CSS. Thư mục web nằm trong thư mục "chess-64-squares-web".
Phong cách của trang web dựng trong theo file: DESIGN.md
Gồm những trang như sau:
* Trang đăng nhập
* Trang đăng đăng kí
* Trang cá nhân (bao gồm danh sách các ván cờ đã chơi)
* Trang đánh cờ: bắt đầu từ tìm trận, vào trận, đánh cờ, kết thúc trận

Backend nằm trong thư mục "chess-64-squares-backend". Sử dụng Nestjs + Postgres + Redis + Socket.io đã xây dựng sẵn các chức năng cơ bản. Hãy đọc kĩ "chess-64-squares-backend". Dựa vào backend để xây dựng web
